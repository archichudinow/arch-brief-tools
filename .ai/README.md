# .ai - AI Context Folder

This folder contains structured documentation for AI assistants
to quickly understand the application architecture and current state.

## Files

- **app-overview.md** - High-level app summary and purpose
- **tech-stack.md** - Technologies, libraries, and dependencies
- **architecture.md** - Application structure and module organization
- **features.md** - Feature list with implementation status
- **data-models.md** - Core data types and interfaces
- **workflow.md** - User flow and step sequence
- **current-status.md** - Development progress and next steps

## Usage

When starting a new session, read these files to understand:
1. What the app does
2. How it's structured
3. What's implemented vs planned
4. Current priorities

This folder is updated after each significant implementation step.
